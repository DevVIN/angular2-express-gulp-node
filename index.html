Angular2 Gulp, Typescript, NodeJS and Express

Introduction

Basic Angular seed application created with Quick start application given on angular website (https://angular.io/docs/ts/latest/quickstart.html). It uses Typescript language and also added support for Express server.

Steps to Run

    npm install
    npm build
    npm start

Dependencies

  1. Angular 2
  2. TypeScript
  3. Gulp
  4. ExpressJS
  5. NodeJS
  6. Nodemon
  7. TsLint
